<p align="center"><em>Anticipez le vocabulaire clé d'une vidéo YouTube, puis visionnez-la avec une compréhension plus fluide et durable.</em></p>

**CaptiPrep = Caption + Preparation.** L'extension transforme les sous-titres d'une vidéo en une phase d'échauffement ciblée : identification des mots/expressions, filtrage assisté par IA, génération de cartes et étude avant le visionnage intégral. Ensuite, le contexte renforce naturellement la mémorisation.

## 🤔 Fonctionnalités
### 1｜Capture automatique des sous-titres & extraction de candidats (mots / expressions).
### 2｜Filtrage intelligent + génération de flashcards (IA en pré‑sélection, vous peaufinez, cartes instantanées).
### 3｜Carnet de vocabulaire (Wordbook) : historique centralisé pour révision et export.
### 4｜Gloss multilingue : apprenez n'importe quelle langue avec des explications dans celle de votre choix.

## ⚙️ Utilisation
1. Installation via Chrome Web Store / Edge Add-ons ou en local (mode développeur > Charger l’extension non empaquetée).
2. Ouvrez une vidéo YouTube sous‑titrée puis cliquez sur l’icône.
3. Flux : extraire → choisir → générer → étudier (Flèches = navigation ; Espace = favori).
4. Ouvrez le Carnet pour revoir à tout moment.
5. Exportez votre vocabulaire consolidé.
6. Modèles conseillés : filtrage gemini-2.5-flash-lite ; cartes gemini-2.5-flash.

## ❓ Pourquoi ça marche
- Pré‑exposition : vous réduisez l’effort cognitif.
- Contexte réel : exemples tirés de la vidéo ciblée.
- Capitalisation : un seul carnet pour tout revoir efficacement.
